How to use a walker

 Front-wheeled Walker
  • Moderate balance deficits
  • Maximal walking stability
  • Can be used to offload weight
  • Amazon
  • Walgreens

How to use four-wheeled walker Four-wheeled Walker (Rollator)
  • Mild balance deficits
  • Minimal walking stability
  • Frequent seated rest breaks
  • CANNOT be used to offload weight
  • NOT to be used as a wheelchair
  • Amazon
  • Bellevue Healthcare

How to use a single-point cane

 Single-Point Cane
  • Mild balance deficits
  • Minimal walking stability
  • Maximal maneuverability
  • CANNOT BE used for offloading weight
  • Amazon
  • Walgreens
  • RiteAid
  • Bellevue Healthcare
How to use a quad cane Quad Cane
  • Mild to moderate balance deficits
  • Better stability than single point cane
  • Useful if user has hemiparesis
  • CANNOT BE used for offloading weight
  • Amazon
  • Bellevue Healthcare

How to use a wheelchair

 Wheelchair
  • Severe balance deficits, pain, or offloading required
  • Maximal stability
  • Best for used for offloading weight
  • Bellevue Healthcare
